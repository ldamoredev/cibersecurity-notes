Attack Surface Mapping Index

Purpose

This index is the root entry point for the attack-surface-mapping branch of the cybersecurity vault.

Use it to: - map what is actually exposed, reachable, and discoverable - connect networking, web-security, API-security, and recon into one operational view - reason about exposure drift, forgotten assets, and undocumented entry points - turn architecture assumptions into observable attack surface

Use Reference Registry — Attack Surface Mapping as the source of truth for references in this branch. Return to Cybersecurity Index for root navigation across branches.
